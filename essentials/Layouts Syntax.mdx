## General tips 

1. Remember to select your input variables if creating a forms.
2. Be very specific and use keywords - the longer the better.
3. Don't use short prompts like `buy button` or `stripe`.

**Examples:** 
```
    Send an SMS via Twilio
```

```
    Verify a user's identity
```

```
    Send data to Salesforce
```
